Instruções Para HTML com CSS


1.Comandos Para diversas utilidades

Target - Serve para direcionar o link de forma diferentes, os dois tipos são o _black e o _self

	_self - Abre o link na mesma página.
	
	_black - Abre o link em outra página.

	Exemplo de uso:

	 <a href="<link>" target="_self">Link para abrir na mesma página</a>

	 <a href="<link>" target="_black">Link para abrir em outra página</a>

Bookmarks - Faz a locomoção pelas páginas extensas ser mais rápida, quando clicar no link, ele vai levar para o local exato da página onde o id com que está nomeado com o link.

	Formula de uso:

		//Não esqueça de usar # no início para informar que é um Bookmarks.
		<a href="#<nome do link>"><Nome que desejar></a>

	Exemplo de uso:

		<a href="#link1>">Noticia1</a>
		<a href="#link2>">Noticia2</a>
		
		<p id = "link1">Noticia Longa<p>
		<p id = "link2">Noticia Longa<p

Download - Força a Hiperligação(Link) a fazer o download do recurso. Ele consegue fazer download de textos, imagens ...

	Exemplo de uso:

		<a href="<link>" download>Baixar</a>
		
		//Aqui baixa o arquivo já com o nome desejado
		<a href="<link>" download = "Nome para o arquivo">Baixar versão com nome</a>

2.Tabelas

	Para criar uma tabela use table, mas para é preciso dos outros elementos para completar. E existe uma orde para criar

	table - Inicia a tabela
	
	thead - Esse é o tiulo para a tabela, ele deixa em negrito informando o que é essa coluna

	tbody - é o conteúdo da tabela, aqui vai td da tabela

	tr - Esse é a linha, deve ser colocado depois do thead ou o tbody, ele é obrigatório para fazer o conteúdo da tabela.

	th - Esse é o conteúdo da tabela do thead, o "titulo" da coluna, ele so deve ser colocado dentro do thead

	td - Esse é o conteúdo normal da tabela que normalmente é colocado no tbody.

3.Formulários

	Para a criação de um formulário no HTML usamos o <form>

	dentro deles podemos usar o input e outros.

Input- O input tem varias funções, com funções diferentes

	formula: <input type=" " name=" ">

	type - é onde você vai escolher a função ou o tipo de informação que o input vai guarda

	name - serve para você colocar o nome para usar com alguma linguagem de programação.

	3.1 - Tipos de input Para Formulários.

		text - Aceita tudo, ele vira uma String

		email - só aceita emails validos, mas não confie, utilize o JavaScript para fazer a validação com segurança.

		passaword - Trata o texto como uma senha, ocultando o texto

		number - só permite números.

		url - Só permite links

		radio - transforma o input em uma opção de marcar OBS: ele vira um tipo de marcação que é necessária outra, ele é uma opção de escolhas 

			(foto de exemplo)

		range - Esse faz uma barra horizontal onde o usuário puxa uma barra azul para preencher.

		submit - Transforma o input em um botão mas é obrigatório que ele esteja em um form.

	3.2 - Atrubutos para por nos Elementos.

		minlength - Da um minimo de caracter que precisa ser inserido.

        maxlength - Da um máximo de caracter que precisa ser inserido.

		min - define o mínimo do números permitido no type number.

        max - define o maximo do número permitido no type number.

        step - permite números com casas decimais no type number.

        required - Faz com que sejá obrigatorio o preenchimento do campo.

    	placeholder - Deixe um texto dentro do campo como se fosse uma dica para o usuário.

		readonly - Impede do texto digitado de ser apagado.

	3.3 - Outros Elementos.

		Há outros elementos que podem ser usados na criação de formulários.

		Label - é o titulo que normalmente é colocado nos campos para identifica-los.
		
		select - Cria um menu de seleção para o usuário.

		textarea - é um campo grande para texto, bem parecido com o input text

		datalist - Ele faz um campo ter varias opções meio que um filtro, ao digitar uma letrar ele reduz  a lista com palavras q possue a letra 				   digitada

4.Funções de clik.

	onclick = "<nome da função do JS ou use um alert("<Texto que desejar>")>" - executa uma função dada por você quando clicado com o botão esquerdo do mouse.

	oncontextmenu = "<nome da função do JS ou use um alert("<Texto que desejar>")>" - executa uma função dada por você quando clicado com o botão direito do mouse.

	onmouseleave = "<nome da função do JS ou use um alert("<Texto que desejar>")>" - executa uma função dada por você quando o mouse entra dentro do elemento.

	onmousemove = "<nome da função do JS ou use um alert("<Texto que desejar>")>" - executa uma função dada por você quando o mouse se move dentro do elemento.

	onmouseleave = "<nome da função do JS ou use um alert("<Texto que desejar>")>" - executa uma função dada por você quando o mouse sai de dentro do elemento.

	Obs: Essas funções só podem ser usadas uma vez no elemento. Para executar mais de um, use o Lista de eventos.

	Lista de eventos
	
	Essa função permite executar varios evenos no mesmo elemento.

	let variavel_exemplo = document.querySelector("<elemento ou id dele>");

	variavel_exemplo.addEventListener("<Evento que voce deseja por exemplo o click(se quiser outro dê um pesquisada)>, <função que você fez ou você pode fazer a função aqui 									  mesmo>") - Com isso você pode repedir esse codigo com diferentes funções que todas vão ser executadas.
