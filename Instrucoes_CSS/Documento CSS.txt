Instruções CSS

BACKGROUND

	
	background-image

		Caso queira colocar uma imagem no background use desta forma:

			background-image: url(<Imagem>);

			Obs: Se você usar o background-size junto, poderá obter resultados interresantes.


	background-size

		Muda o tamnho interno do fundo usando vários atribitos como auto, couver e outros. Explore para aprender melhor. 

	Display
	
		Essa propriedade determina como um elemento vai se comportar, se vai quebrar uma linha ou ficar um elemento do lado do outro.

		display: inline; - Impede do elemento quebra a linha

		display: block; - Obriga os elementos a quebar a linha.

		display: inline-block; - esse é os dois, n quebra linha e aceita quebrar se colocar largura(widtch)

		display: flex; - Obriga os elementos a quebrar a linha e organiza eles lado a lado. Obs: Essa função é importantíssima para fazer funções que usam flex 			 funcionar, ela é obrigatória.

		display: grid; - Organiza os elementos dentro de um contêiner de forma mais controlada.  Obs: Essa função é importantíssima para fazer funções que usam 				 grid funcionar, ela é obrigatória.

		display: none - Faz o elemento nem existir, pode ser útil junto ao Media Queries

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
Obs: Esses Elemento abaixo precisa da função display: flex; para funcionar.

	Flex

		Essa propriedade torna os elementos flexíveis de varias maneiras.

		flex-diretion - Ajeita os elementos, podendo deixar em linhas usando o column ou deixando em colunas usando o row. Mas ele só faz ficar um do lado do 				outro, pare ele colocar abaixo também, use flex-warp junto.

		flex-warp - Ele só funciona junto com o flex-diretion, ele deixa os elementos do flex-diretion um abaixo do outro, deixando em ordem.

		flex-flow - Ele é as funções de flex-diretion e flex-warp juntas, apenas diga o que você quer e não coleque virgulas entra elas.

		.Essas outras funções poda dar um charme ou efeito no seu código.

		flex-grow - Ele faz o elemento preencher todo o espaço do contêiner, deixando espaço apenas suficiente para os outros, ele aumenta a escala.

		flex-shrink - Ele faz o elemento diminuir, ele diminui a escala.
		

			 Obs: Você pode inverter a posição dos elemento colocando um reverse apois eles.
		

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

Obs: Esses Elementos abaixo precisa da função display: Grid; para funcionar.

	Grid
		obs:Você pode usar também o auto, para fazer o alinhamento automático

		grid-template-columns - Comportamento deitado, de forma horizontal, coloque as porcentagens que cada elemento vai ocupar, faça isso sem usar virgulas.

		grid-template-rows - Comportamento de Coluna, de forma vertical, coloque as porcentagens que cada elemento vai ocupar, faça isso sem usar virgulas.
		
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
Obs: Essas funções para poder funcionar é necessário um display:flex; ou um display:grid;, As funções que funcionam apenas com um desses displays está sinalizado.

	Justify-content

		Usado para alinhamento entre os elemento, por exemplo no center(Centro)
		
		Outras funções:

			space-<evenly/ around/ between> - Faz um espaçamento vertical igual entre os elementos, faça eles para ver as diferenças.

			Obs: É necessário usar um display:flex no elemento para essa função funcionar.

	Align-items 

		Usado parra fazer alinhamento dos elementos de diversos jeitos.

		Center - centro

		flex-start - Coloca os elementos no topo do contêiner.

		flex-end - Coloca os elementos na parte inferior do contêiner.

		Stretch - Coloca todos os elementos na altura do elemento com mais conteúdo.


	Obs: Essa função só funciona com o display:grid;

	Place-items 

		Movimenta o elemento verticalmente e horizontalmente, 

		Formula - place-items: <escolha a direção vertical> <escolha direção horizontal>;

	Gap

		Usado parra dar espaçamentos entre elementos.

		gap - Espaçamento em todas as direções.

		column-gap - Espaçamento lateral.

		row-gap - Espaçamento inferior.

	order 

		Use para poder dar uma ordem nos elementos, o elemento que tiver o maior numero no order irá fica na frente.

	Float

		Usado para alinhar imagens com textos.

		Formula - float:<Escolha a direção>


	Position

		Serve para mexer no posicionamento do elemento, podendo deixar impossível de tiralo do lugar.

		static - Deixa o elemento estático no lugar, impedindo de qualquer outro elemento empurralo.

		relative - Os outros elementos vão tratar o elemento que possue esse atributo como background, ignorando e nem mexendo nele.

		absolute - Deixa o elemento por cima de tudo, é como se ele não estivesse mais no código original e sim em outra camada.
		
		fixed - Deixe o elemento fixo na tela, mesmo se rolar a tela ele vai continuar no lugar que foi colocado. 


	Gradiente

		Caso queira usar esse tipo de background você terá que usar o comando dessa forma:

			background: linear-gradient(80deg, red, yellow);

			Explicação:

				1° - É a rotação das cores em graus, mude o numero entre 0 e 360.
				
				2° - Primeira cor.

				3° - Segunda cor.

	Z-index 

		O z-index da prioridade no elemento, caso tenha eleentos um em cima do outro, o que estiver com o z-index maior, ficará na frente.


TEXTOS no CSS

	Alinhamento Horizontal de Textos.

		Para alinhar os textos usamos o atributo, usamos o text-align
	
		Formula - text-align: <Onde deseja alinhar)>

		Há varias maneiras de alinhar os textos aqui vai as mais utilizadas
	
			center -  Alinha ao centro.

			left - Alinha a esquerda

			rigth - Alinha a direita

	Espaçamento entre Letras e Palavras

		Para dar espaçamento entre as linhas ou palavras usamos:
	
		line-height - Espaçamento entre linhas.

		leeter-spacing - Espaçamentre entre letras.

	Shadow(Sombras)

		Para aplicar sombra em textos, use text-shadow

		Formula - text-shadow: 2px 2px 10px rgb(0,0,0,0.5)

			Cada casa é uma configuração da sombra

			1° - Distancia horizontal da sombra em relação ao texto
			
			2° - Distancia vertical da sombra em relação ao texto

			3° - Desfoque do sombra

			4° - Cor

			Com isso você consegue fazer vários tipos de sombreamento, apenas mexa!

	Text-transform - Pode facilitar automatizando aparecimento de textos.

	text-transform: uppercase - Transforma o texto todo em letra maiúscula.

	text-transform: lowercase - Transforma o texto todo em letra minúscula.

	text-transform: capitalize - Transforma a primeira letra de cada palavra em maiúscula.
	

	Fonts

Pseudo classes

	Tras funções automáticas e uteis para ajudar no css.

	:hover - Se o mouse estiver em cima.

	:nth-cild(<numero da ordem do elemento do html>) - Seleciona um elemento através do ordem que ele está em relação ao elemento pai.


Media Queries

	Pode controlar/executar as funções em diferentes tamanhos de tela

	 @media screen and (<Tamnho em height(Altura) ou em width(Largura) da tela em px>) - Serve para executar comandos quando o tamanho da screen(Tela) está 	  										     inferior ou superior ao valor dado

	@media screen and (orientation: landscape(Deitado)/ portrait(Em pé);) - Serve para verificar se o aparelho(Geralmente mobile) está no modo deitado ou em pé e 										executa
